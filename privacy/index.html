Privacy Policy

Last updated: March 2026

Overview

How we handle your data.

No. This website does not use cookies, tracking scripts, analytics services, or any data collection mechanisms that we control.

We do not collect, store, process, or sell any personal data. We do not require accounts, logins, or any personally identifiable information.

All website assets (JavaScript libraries, CSS frameworks, fonts, images) are self-hosted and served from our own domain. We do not load any scripts, fonts, or trackers from external CDNs or third-party servers (with the exception of the CoinGecko widget, documented below).

This website is hosted on GitHub Pages and uses the following external services:

  • Cloudflare - DNS and content delivery (see below)
  • CoinGecko Widget - Cryptocurrency price display
  • GitHub Pages - Website hosting

Each of these services may collect data according to their own privacy policies. We have no control over Cloudflare's data practices. Links to their privacy policies are provided below.

External links: This website contains links to external sites (blockchain explorers, exchanges, social media, wallets). These are regular links that users choose to click. No data is sent to these sites until you click them. Each has its own privacy policy.

This domain (blackcoin.org) uses Cloudflare as its DNS provider. Cloudflare sits between the website and visitors, providing:

  • DNS resolution
  • DDoS protection
  • Content delivery network (CDN)
  • Performance optimization

We do not have control over Cloudflare's analytics or data collection. Cloudflare may collect analytics data, serve scripts, or track visitor behavior according to their own policies. This is outside our control.

If you block Cloudflare's beacon.min.js script in your browser, the website will continue to function normally. The site itself does not require Cloudflare to work - it only needs DNS to resolve.

This privacy policy applies only to blackcoin.org and blackcoin.dev. Other websites related to Blackcoin (such as blackcoin.nl, blackcoinmore.org, exchanges, social media) have their own privacy policies and data practices.

The official Blackcoin websites are:
  • blackcoin.org (this site)
  • blackcoin.dev
  • blackcoinmore.org
  • blackcoin.nl

We are not responsible for the privacy practices of any other website.

Third-Party Services

Services used by this website and their privacy practices.

What it does: Cloudflare provides DNS (Domain Name System) resolution and acts as a proxy between visitors and GitHub Pages where this site is hosted.

Data collected: Cloudflare may collect IP addresses, browser information, request logs, and analytics data. They may also inject scripts (like beacon.min.js) for analytics.

Our control: None. The domain uses Cloudflare nameservers, but we do not have access to the Cloudflare account controlling them.

Privacy policy: cloudflare.com/privacypolicy

How to opt-out: Use browser extensions that block Cloudflare scripts, or use privacy-focused DNS resolvers.

What it does: Displays live cryptocurrency price data for Blackcoin (BLK) in the navigation menu.

Data collected: CoinGecko may collect IP addresses and usage statistics for the widget. No personally identifiable information is required to view the price data.

Our control: We chose to include this widget. It loads from widgets.coingecko.com.

Privacy policy: coingecko.com/privacy

How to opt-out: Block widgets.coingecko.com in your browser or use a script blocker.

What it does: Hosts the website files and serves them to visitors through GitHub's global CDN.

Data collected: GitHub Pages may log IP addresses, request data, and browser user agents for technical purposes. These logs are typically not persistent.

Our control: We host the site on GitHub Pages and can modify the content at any time.

Privacy policy: GitHub Privacy Statement

Note: GitHub Pages does not inject tracking scripts into your website.

What it does: The source code for this website is publicly available on GitHub. Anyone can view, fork, or contribute to the codebase.

Repository: github.com/CoinBlack

Data collected: GitHub may collect data about who visits the repository (logged-in users). The website code itself does not contain any tracking code.

Transparency: All website code is open source and can be audited by anyone. We removed Google Tag Manager, Twitter widgets, and other tracking scripts in March 2026.

This website contains links to external sites:
  • Cryptocurrency exchanges (XeggeX, FreiXlite, etc.)
  • Blockchain explorers (CryptoID, CoinMarketCap, etc.)
  • Social media (X/Twitter, Discord, Telegram)
  • Wallet downloads (Coinomi, Komodo)

We are not responsible for the privacy practices of these external sites. Each has its own privacy policy. We recommend reviewing their policies before providing any personal information.

Wallet downloads are hosted directly on this website and GitHub.

What we host: Wallet executables (.zip files) and documentation (.pdf files) in the assets/ directory.

Data collected: None. Downloads are served directly from GitHub Pages without any tracking or analytics.

Security: We recommend verifying checksums and downloading only from official sources. The only official Blackcoin websites are listed in our FAQ.
